# Alto Multimodal Trail

Static advocacy website for the Alto Multimodal Trail concept.

## Stack

- GitHub Pages
- Jekyll
- Markdown content pages
- Shared layouts/includes
- Custom CSS

## Local development

1. Install Ruby and Bundler.
2. Run `bundle install`.
3. Run `bundle exec jekyll serve`.
4. Open the local server URL shown in the terminal.
